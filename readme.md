# Paper Data Repository

This repository holds datasets and supporting files related to business ethics research. It includes raw/processed data, tabular summaries, and visual materials used in experiments or analysis.

## Directory Overview

- `data/` – Core datasets used in the study (raw, processed, or aggregated data files).
- `excel/` – The core dataset (raw data from WoS) used in the research.
- `pic/` – Figures, charts, and other image assets that illustrate experimental results or findings.
- `BE.ipynb` – Notebook used to explore, analyze, or visualize the data for the paper.

Feel free to explore each folder for more specific details on the data and related processing steps.
